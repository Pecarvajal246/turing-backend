# Turing Backend


<img src="./assets/turing-logo.jpeg" alt="Turing Wallet" style="height: 150px; width:150px;"/>

## Installation of dependencies


```
npm i
```

## Run serverless


```
ntl dev
```

## Assign the environment variables in the .env file

```
# APP
APP_MODE=development

# MONGO
MONGODB_URL=

# EMAIL
EMAIL_PASS=

# BINANCE
BINANCE_KEY=
BINANCE_SECRET=

# JWT
SECRET_TOKEN=

# TWILIO
TWILIO_ACCOUNT_SID=
TWILIO_AUTH_TOKEN=
TWILIO_PHONE=

# MJML
MJML_ID=
MJML_SECRET=
```
